Welcome

This collection of examples for scientific computing contains completely worked solutions for beginners in numerical methods. We use the programming language Python and the extension packages numpy and scipy. The examples are provided as IPython notebooks.

The examples are taken from the following sources:

  • Exempelsamling i numeriska metoder by Lennart Edsberg, Gerd Erikson and Bengt Lindberg. You find the brochure online in Swedish language. We provide an English translation in each exercise. The translations are sometimes a bit shortened or simplified if they can be understood also in that way,
  • Numerical Analysis (2nd. ed.) by Timothy Sauer.
